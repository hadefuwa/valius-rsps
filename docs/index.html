Welcome to the Valius Wiki

Work in progress. This wiki is actively being built. More articles coming soon — contributions welcome via GitHub.

Valius is a 317-revision Old School RuneScape private server with custom content, PvP systems, and a growing economy. This wiki documents game mechanics, items, NPCs, and server-specific features to help both new and experienced players.

Quick Start

New to Valius? Here are the essentials to get up and running:

Step Details
1. Start the Server Requires Java 11 (Amazon Corretto). Run gradlew.bat run from
Valius-Server-master\ — listens on port 43594
2. Launch the Client Must use Java 8 (JavaFX is bundled in JDK 8 only).
Run: java.exe -jar client.jar
3. Version Match Config.CLIENT_VERSION = 5 must match on both server & client.
A mismatch causes silent login failure with no error shown. Critical
4. Game Cache Cache lives at C:\Users\[you]\ValiusCache1\ — already extracted.
Do not delete this folder.

Blood Money

Blood Money (item ID: 13307) is the primary PvP reward currency in Valius. Earned by defeating other players in the wilderness or PK District, it can be exchanged at the Point Reward Shops NPC at the home teleport for powerful custom gear including Goliath Gloves, Swift Gloves, Spellcaster Gloves, and more.
