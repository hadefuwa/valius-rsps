Valius is a 317-revision Old School RuneScape private server with custom content, PvP systems, and a growing economy. This wiki documents game mechanics, items, NPCs, and server-specific features to help players.

Blood Money

Blood Money (item ID: 13307) is the primary PvP reward currency in Valius. Earned by defeating other players, it can be exchanged at the Point Reward Shops NPC at home for powerful custom gear including Goliath Gloves, Swift Gloves, Spellcaster Gloves, and more.

Developer Reference Developers

Dev docs. This section covers running, configuring, and modifying the Valius server. Switch to the 👤 Players tab for gameplay guides.

Valius is a Java-based 317 RSPS. The server runs on Gradle + Java 11; the client requires Java 8 for bundled JavaFX. Both live in the same GitHub repo under separate subdirectories.

Running the Server

ComponentCommand / Notes
Server Java 11 required (Amazon Corretto). Port 43594.
set JAVA_HOME=C:\Program Files\Amazon Corretto\jdk11.0.29_7
gradlew.bat run
Client Must use Java 8 — JavaFX is bundled in JDK 8 only.
"C:\Program Files\Java\jre-1.8\bin\java.exe" -jar client.jar
Rebuild client set JAVA_HOME=... then gradlew.bat jar in Valius-Client-master\
Game cache C:\Users\[you]\ValiusCache1\ — already extracted, do not delete.
Player saves Valius-Server-master\Data\saves\

Critical Configuration

Critical  Config.CLIENT_VERSION = 5 must match on both server and client. A mismatch causes silent login failure — no error is shown in-game or in logs.
